This source code implements the Sequential Interval Network framework (SIN) in the following paper:

From Stochastic Grammar to Bayes Network: Probabilistic Parsing of Complex Activity. 
Nam N. Vo and Aaron F. Bobick. CVPR 2014

To use it, add the "sin" folder and all its sub folders to Matlab path.
Examples of how to use the code are in the "examples" folder.
Note that data and code for experiments reported in the paper are not included and have to be downloaded separately.

In case you use this code, consider citing our paper. 
Contact us at: namvo@gatech.edu if you find any bugs.
